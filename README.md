# Tax Property Manager (Prototype)

เว็บต้นแบบสำหรับทะเบียนภาษีป้าย และภาษีที่ดิน/สิ่งปลูกสร้าง โดยทำงานแบบ Static Web App จึงเปิดบน GitHub Pages ได้ฟรี

## สิ่งที่ทำได้แล้ว

- Dashboard ยอดภาษี/สถานะงาน
- เพิ่มและแก้ไขทะเบียนภาษีป้าย พร้อม URL รูปภาพ ขนาด จำนวน หมวด อัตรา และยอดภาษี
- เพิ่มและแก้ไขทะเบียนภาษีที่ดิน พร้อมเลขโฉนด พื้นที่ การใช้ประโยชน์ และราคาประเมิน
- ค้นหา/กรองสถานะป้าย และส่งออกเป็น CSV (เปิดด้วย Excel ได้)
- เชื่อม Firebase Firestore เพื่อเก็บข้อมูลกลางและอัปเดตข้ามเครื่อง (มี Local Storage เป็นข้อมูลสำรอง)

## ใช้งานฟรี

1. สร้าง GitHub repository แล้วอัปโหลดไฟล์ทั้งหมด
2. ไปที่ **Settings → Pages → Deploy from a branch → main / root**
3. เปิด URL ที่ GitHub สร้างให้

## ตั้งค่า Firestore ก่อนใช้ข้อมูลจริง

ใน Firebase Console ไปที่ **Firestore Database → Rules** และเผยแพร่ Rules ชั่วคราวนี้เพื่อให้เว็บต้นแบบสามารถอ่าน/เขียนข้อมูลได้:

```text
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /taxData/{document} {
      allow read, write: if true;
    }
  }
}
```

Rules ชุดนี้เหมาะสำหรับช่วงทดลองเท่านั้น เพราะทุกคนที่มีลิงก์สามารถแก้ไขข้อมูลได้ ควรเพิ่ม Firebase Authentication และจำกัดสิทธิ์ก่อนใส่ข้อมูลจริง.

> หมายเหตุ: สูตรและอัตราในต้นแบบเป็นข้อมูลตัวอย่าง ต้องตรวจสอบกับประกาศและการประเมินของหน่วยงานท้องถิ่นของปีภาษีนั้นก่อนใช้งานจริง
